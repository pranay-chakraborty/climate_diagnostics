💡 **Prerequisites**
===================

Before starting these tutorials, make sure you have:

- Python 3.11+ installed
- Basic familiarity with NumPy and xarray
- Climate Diagnostics Toolkit installed
- Sample climate data (we'll show you how to get some!)
